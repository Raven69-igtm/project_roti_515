01 — Pendahuluan

Latar Belakang

Perkembangan teknologi informasi membuka peluang besar bagi UMKM untuk memperluas jangkauan pasar dan meningkatkan efisiensi operasional secara signifikan.

Kondisi Terkini Roti 515

Roti 515 merupakan UMKM bakery lokal di Nganjuk yang selama ini masih mengandalkan pemesanan secara manual melalui tatap muka maupun pesan singkat tanpa sistem yang terstruktur.

Pemesanan Manual Tidak Terstruktur

Tidak ada konfirmasi otomatis, berisiko miskomunikasi dan kehilangan data pesanan dari pelanggan.

Tidak Ada Pencatatan Data yang Sistematis

Admin kesulitan memantau stok, tren penjualan, dan riwayat transaksi secara akurat dan real-time.

Solusi yang Dihadirkan
  • Aplikasi mobile Android dan Web terintegrasi
  • Pemesanan daring dengan konfirmasi otomatis
  • Notifikasi status pesanan real-time
  • Dashboard manajemen stok dan analitik
  • Backend Golang berkinerja tinggi
  • Database MySQL terstruktur
Tujuan Pengembangan
  • Meningkatkan efisiensi operasional UMKM
  • Memperluas jangkauan pelanggan secara digital
  • Menyediakan data analitik bisnis akurat
  • Meningkatkan kredibilitas dan citra toko

DatabaseMySQL 8.xPenyimpanan data relasional terstruktur
AutentikasiJWT + Google OAuth 2.0Manajemen sesi dan otorisasi pengguna
Hosting APIRailway CloudDeployment server Go dan database MySQL
Hosting WebVercelLanding page, CDN global, distribusi APK
Keunggulan Backend Golang
  • Performa tinggi dengan model concurrency goroutine
  • Kompilasi ke binary tunggal tanpa runtime tambahan
  • Konsumsi memori sangat efisien dibanding Node.js
  • Strongly typed, meminimalkan runtime error di production

12 — Penutup

Kesimpulan

Capaian Sistem
  • Aplikasi tersedia di Android (APK) dan Web Browser
  • Autentikasi dua role Customer dan Admin berbasis JWT
  • Pemesanan dan notifikasi status berjalan secara real-time
  • Dashboard admin dengan grafik penjualan interaktif
  • Backend Golang di Railway, frontend di Vercel
  • Database MySQL terkelola dengan skema relasional terstruktur
Manfaat bagi UMKM Roti 515
  • Efisiensi proses pemesanan dari manual ke digital
  • Data penjualan dan stok tercatat otomatis di MySQL
  • Jangkauan pelanggan lebih luas tanpa batasan geografis
  • Meningkatkan kredibilitas dan citra profesional toko
Rencana Pengembangan Lanjutan
  • Integrasi payment gateway (Midtrans atau DOKU)
  • Fitur ulasan dan penilaian produk oleh pelanggan
  • Push notification via Firebase Cloud Messaging
  • Laporan keuangan dengan ekspor PDF and Excel
  • Publikasi ke Google Play Store
  • Migrasi ke arsitektur microservice berbasis Go
