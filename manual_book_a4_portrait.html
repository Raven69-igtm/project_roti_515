ROTI 515 / PENGANTAR

01. PENDAHULUAN

Roti 515 hadir sebagai simbol transformasi digital bagi UMKM bakery. Kami tidak hanya menjual roti, kami menjual pengalaman belanja yang modern, praktis, dan terpercaya bagi setiap pelanggan.

VISI & MISI

Dimulai dari keinginan untuk membantu toko roti lokal merambah dunia digital, aplikasi ini dikembangkan untuk menjembatani antara tradisi rasa dan teknologi masa depan dengan hanya satu sentuhan.

AKSES APLIKASI

Web App: https://roti515.up.railway.app

Mobile App: Download dan instalasi file APK via Android device.

ROTI 515 / PANDUAN PELANGGAN

02. EKSPLORASI APLIKASI

A. REGISTRASI & LOGIN

1. Buka aplikasi Roti 515.

2. Pilih menu Login jika sudah memiliki akun, atau Daftar jika pengguna baru.

3. Masukkan alamat email dan password.

4. Setelah berhasil, Anda akan diarahkan ke Beranda.

Fig 1: Layar Autentikasi

B. MENCARI PRODUK

1. Di halaman utama, lihat bagian Bestsellers.

2. Gunakan pencarian atau filter kategori untuk mencari roti spesifik.

3. Ketuk foto produk untuk melihat deskripsi, harga, dan stok.

Fig 2: Beranda & Katalog

PANDUAN PELANGGAN PAGE 02
ROTI 515 / TRANSAKSI

03. PEMESANAN

C. MELAKUKAN PESANAN

1. Pada halaman detail produk, atur kuantitas pesanan.

2. Tambahkan catatan opsional jika diperlukan (contoh: "dipisah plastiknya").

3. Tekan tombol Tambah ke Keranjang.

Fig 3: Detail Produk

D. PROSES CHECKOUT

1. Buka menu Keranjang (Cart) di navigasi bawah.

2. Periksa kembali rincian pesanan.

3. Pilih metode pembayaran dan jadwalkan pengambilan.

4. Tekan tombol Checkout untuk mengonfirmasi.

Fig 4: Halaman Keranjang

TRANSAKSI & CHECKOUT PAGE 03
ROTI 515 / MANAJEMEN PESANAN

04. STATUS PESANAN

E. MELACAK & MEMBATALKAN PESANAN

1. Buka menu Riwayat Pesanan untuk melihat pesanan (Menunggu Pembayaran, Diproses, Siap Diambil).

2. Ketuk pada salah satu pesanan untuk melihat detail struk/invoice elektronik Anda.

3. Untuk membatalkan, pilih pesanan berstatus Menunggu Pembayaran, scroll ke bawah, lalu tekan tombol Batalkan Pesanan.

Catatan: Pesanan yang sudah mulai "Diproses" oleh Admin tidak dapat dibatalkan melalui aplikasi.

05. PANDUAN ADMIN

A. DASHBOARD & STATISTIK

Admin memiliki panel khusus untuk melihat pergerakan penjualan, pesanan masuk terbaru, dan pendapatan harian. Admin wajib mengawasi status pesanan di sini.

Menu kelola produk digunakan untuk menambah varian roti baru, memperbarui harga, hingga mengatur stok produk secara real-time yang langsung tampil di layar pelanggan.
